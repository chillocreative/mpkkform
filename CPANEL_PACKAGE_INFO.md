## 🚀 Quick Deployment Steps

1. **Download** `MPKK-Form-cPanel-Complete.zip`
2. **Upload** to cPanel File Manager
3. **Extract** in your desired directory
4. **Create** MySQL database in cPanel
5. **Import** `schema.sql` via phpMyAdmin
6. **Configure** `.env` file with database credentials
7. **Setup** Node.js app in cPanel
8. **Install** dependencies (npm install)
9. **Start** the application
